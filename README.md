# SecKill

---

## Introduction
秒杀系统，实现秒杀功能。

---

## Run
1. 导入schema.sql和seckill.sql
2. 修改jdbc.properties数据库配置连接
3. 运行Tomcat
4. 在浏览器里面输入http://localhost:8080/seckill/list
5. Enjoy it！

---

## Environment
- IDEA
- JDK1.8
- Maven
- MySQL
- Redis
